# PS2 Memory Card Manager

A simple GUI application for managing PlayStation 2 memory card files (.ps2, .mcd).

## Features

- View and inspect PS2 memory card contents
- Browse save files on memory cards
- Inspect individual save file details

## Requirements

- Python 3.x
- PySide6 >= 6.5.0
- mymcplusplus >= 0.1.0

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the application:
```bash
python main.py
```
